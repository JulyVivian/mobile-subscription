# Justify

## Overview

文本两端对齐，用于将一串文字两端对齐显示。

是否全局组件：否。

## Usage

```html
<justify text="123456789"></justify>
```

## Slots

| Name | Description | Default |
| ----- | ----- | ----- |
| default | 显示的文本内容 | {{text}} |

## Props

| Property | Type | Description | Default |
| ----- | ----- | ----- | ----- |
| text | String | 显示的文本内容 | "" |
